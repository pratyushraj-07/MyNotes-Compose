# MyNotes-Note-app
# Description
- A simple notes app built using Kotlin, Jetpack Compose and Room DB.
- Includes animations for:
  - Navigation.
  - Note deletion.
- The primary goal of this project was to practice and implement:
  - Room DB.
  - MVVM architecture.
  - Basic navigation.
  - Animations in Android development.
